第一次运行首先安装依赖，只需要运行一次

```bash
npm insall
```

然后本地启动项目

```bash
npm run dev
```

### 测试

```bash
npm test
```


### 运行环境

本项目在macOS Sierra 10.12.6 或更高版本 以及 Chrome 68或更高版本下运行，其他操作系统和浏览器尚未测试。

### 打包以及上线

先将程序为生产环境打包
```bash
npm build
```

```bash
npm start
```
该命令用于在云服务器里直接运行，本地可以用于测试

```bash
lean deploy
```
